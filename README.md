**☆ 注意 ☆**

>   **`1.3.0`开始，全面支持`AndroidX`，弃用`support`**

### 用法

* `build.gradle`中添加以下依赖

  ```groovy
  dependencies {
      compile 'com.yhy.utils:andr-utils:latestVersion'
  }
  ```
